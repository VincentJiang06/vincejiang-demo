本站支持开放接口 —— 给 Agent / 程序直接抓

这页的数据来自一个开放的只读接口(已开 CORS、无需鉴权),拿去喂给你的 agent / 脚本即可:

GET https://vincejiang.com/_status/api

返回 schema: "vincejiang.status/1",核心字段:

  • providers[].indicator — 官方级别:none/minor/major/critical/maintenance
  • providers[].healthy — 是否正常(true=一切正常;常驻通知不影响它)
  • providers[].description — 官方英文原状态
  • providers[].summary_zh — DeepSeek 改写的通俗中文解说
  • providers[].incidents[] — 进行中的真故障(含 duration 持续时间)
  • providers[].notices[] — 常驻通知(如某模型被暂停,不算故障,含 name_zh)

给你的 agent 一段现成 prompt(点下面按钮复制即可用):

你是 AI 服务状态助手。请用 HTTP GET 拉取 https://vincejiang.com/_status/api(开放 JSON、无需鉴权,schema 为 vincejiang.status/1)。然后对 OpenAI 和 Claude 各汇报一行:① 是否正常(healthy=true 即正常);② 官方级别 indicator;③ 直接引用 summary_zh 作为中文解说;④ 若 incidents 非空,补一句故障名与 duration.text 持续时间。注意:notices 是常驻通知、不算故障,默认忽略。用简洁中文回答。

说明:翻译只在官方状态发生变化时才重新生成(其余时刻走缓存),所以接口又新又省。
