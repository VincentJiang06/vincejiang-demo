# reactor-study 课程文风契约（草料文风适配版）

> 提取自 caoliao-style-writer v3（`~/playground/skill-developer/Caoliao-Docs/caoliao-style-writer/`，docs/voice.md + law.md + philosophy.md + control/rules.yaml），按课程网站语境适配。改写与校验共用本契约。底盘一句话：**微信的密度，飞书的体贴，草料的诚实。**

## 0. 读者契约（课程版）

目标读者：**会写 eval 的工程师 / 关心排名与测量机制的自学者**。识字、看得懂论文引用、讨厌被灌水，也讨厌被端着教育。两条容忍边界：
- **下边界（俚语线）**：小玩意儿/搞定/说白了/挨个/一来一回/咱们 → 读者嫌掉价。
- **上边界（空书面线）**：赋能/闭环/范式/宏大叙事空转/"处于…状态" → 读者嫌空。
- **目标区**：有画面的具体白描 + 精确机制 + 硬引用。判据：**这句像不像一个懂行的人在具体讲一件事给同行听。**

## 1. 九条红线（课程版，error 级）

1. **开门见山**：每节第一段直接讲这节的核心命题（什么现象+什么机制+什么结论），痛点/故事一句带过，不做开场主角。
2. **禁元叙事**："下面我们先…再…""接下来的三条谱系是…"删掉，结构靠标题与目录自明。
3. **禁修辞设问**：正文与标题不用设问句造势。唯一例外：文末"留一个问题"小节的真问题（FAQ-Q 性质）可以带问号。
4. **禁对仗排比套话**（AI 味最重）："作得了数、查得到"式短语成对排比 → 改具体可验证陈述。
5. **破折号「——」全篇 0**：补充说明改冒号、插入语改逗号、转折改分句。一处不留。
6. **价值由机制带出**：先"是什么/怎么运转"再"意味着什么"；形容词克制，删掉后句子仍成立才合格。
7. **因果链三件套**：每个论点 = 失败模式（具体）→ 机制（怎么发生/消解）→ 边界（成立条件、证据强度）。
8. **禁无支撑泛化**："整个行业都在…""众所周知…" → 删或换成带出处的陈述。
9. **数字与引文有源**：年份/被引数/实验数值全部可对回 `research/` 或 `research/deep/`；报告里标【存疑】的，课文要么不用、要么写明"待核"。这是课程版的**零知识注入**：不写研究报告里没有的事实。

## 2. 语气与句式

- 人称统一「你」；建议式（可以/值得/如需），不居高临下也不讨好。
- **书面动词优先**：测量、优化、坍缩、泄漏、校准、约束、放大、稀释、对冲、归因、复核。
- **逻辑连接词撑密度**：因此/从而/于是/一旦/只要/即可/无需/反之。这是微信 register 最强签名，比删口语更重要。
- 短句一句一件事；长枚举拆列表；概念对比用表格。
- 比喻纪律：功能性比喻（尺子/考场/靶子）可用，一处点到即止；不铺排、不抒情、不"同一只野兽的三张皮"式叠喻。
- 卖萌语气词（啦/呀/哦/嘛）零出现；感叹号约等于零。

## 3. 诚实边界（课程的身份，比草料原版更狠）

- 每个强结论带成立条件："X 成立，边界是 Y"。
- 证据分级明写：一手实锤 / 二手转述 / 存疑待核，跟随 D 报告的标注。
- 反方证据不藏：命题哪里不成立、哪条证据被打折，正文明说（这是本项目区别于科普号的身份）。
- 纠讹是招牌：凡涉及流行讹传（Strathern/眼镜蛇/Hawthorne/四机制…），给正确出处，语气平实不猎奇。

## 4. 结构件约定（lesson blocks）

- `prose` 开篇段 = 红线 1 的"开门见山段"。
- `callout(intuit)` = 一段话讲透一个直觉，机制语言，不设问。
- `callout(myth)` = 纠讹卡：流行说法 → 复核结论 → 一手出处。
- `callout(applied)` = 写给做 eval / 看排名的你：具体操作含义。
- `module` 标题与内部 UI 文案同契约（按钮/滑杆/读数标签里不许有俚语和破折号）。
- `sources` = 作者·年份·标题·出处；深化报告的补充源写 `research/deep/Dx`。
- 文末可加一段"留一个问题"：从 D 报告开放问题里取，真问题、可带问号。

## 5. 机械禁词表（继承 rules.yaml，error）

- 吹嘘：强大/海量/业界领先/极致/赋能/助力/打造/一站式/全方位/全面提升/颠覆/重磅
- 空书面：处于…状态/一体化闭环/可视化管控/全域覆盖/协同联动/工作抓手/提质增效
- 俚语：活儿/搞定/糊弄/犄角旮旯/老毛病/咱们/说白了/挨个/管够/稳稳/少走弯路
- 软口语：其实吧/真的很/超级/省心/省事/顺手/随手/摸清
- AI 指纹：破折号——密集/对仗排比/设问开场/"值得注意的是"/"让我们"

## 6. 校验流程

写完一枝 → `node site/tools/style-gate.mjs`（若建）或人工按 §1/§5 过一遍 → Opus 校验轮按本契约逐节挑刺（只报违例+改法，不重写事实）。
